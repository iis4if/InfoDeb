# InfoDeb
macOS app for show package info a deb!

### Features

* Show deb package info 
- [ ]  Name
   - [ ]  PackageName
   - [ ]  Package
   - [ ]  Section
   - [ ]  Version
   - [ ]  PackageAuthor
   - [ ]  Description
   - [ ]  PackageSize
   - [ ]  InstalledSize
   - [ ]  PackageMaintainer
   - [ ]  PackageArchitecture
   - [ ]  PackageIcon
   - [ ]  PackageFileName
   - [ ]  PackageSileoDepiction
   - [ ]  PackageDepiction
   - [ ]  md5
   - [ ]  sha1
   - [ ]  sha256
   - [ ]  Depends
   - [ ]  Conflicts
   - [ ]  Replaces
   - [ ]  Provides

* Encrypt deb file to 
- [ ]  md5
- [ ]  sha1
- [ ]  sha256
